## 🚀 How to Run Locally
1. Clone this repository: `git clone https://github.com/KANOK508/hero-dev-portfolio-p01.git`
2. Navigate to the folder: `cd hero-dev-portfolio-p01`
3. Install dependencies: `npm install`
4. Run the development server: `npm run dev`
